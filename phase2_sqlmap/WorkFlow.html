Component Details

1. Docker Check (orchestrator.py)

Checks if Kali container is running and starts DVWA, Juice Shop, Neo4j and Kali containers.

2. Fetch Neo4j (Graph RAG)

Queries Neo4j to retrieve endpoints discovered by the Recon Agent.

3. SQLMap Planning (LLM Agent)

Uses Ollama LLM (Llama3 / DeepSeek) to generate SQLMap commands based on discovered endpoints.

4. SQLMap Execution

Executes generated commands inside the Kali container and captures output and logs.

5. Vulnerability Detection

Analyzes SQLMap output to detect SQL injection indicators.

6. Store Results

Stores the complete workflow in Neo4j for traceability and graph visualization.
